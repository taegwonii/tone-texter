# Icons

Place your extension icons here:
- `icon16.png` — 16×16 px (toolbar)
- `icon48.png` — 48×48 px (extensions page)
- `icon128.png` — 128×128 px (Chrome Web Store)

Until you add real icons, Chrome will show a generic placeholder — that's fine for development.

## Design tips
- Use a simple, recognizable shape (speech bubble, chat icon, etc.)
- Keep the background transparent or use brand color (#6366f1)
- Test at 16px — details get lost at small sizes
